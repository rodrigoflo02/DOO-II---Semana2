Descripcion del proyecto
El proyecto es una continuacion del proyector anterior para la empresa SpeedFast, en la que se mejora la estructura haciendola mas robusta y reutilizable

Estructura general del proyecto
El proyecto esta estructurado en 2 paquetes: Model que contiene las clase base abstracta Pedido y las subclases PedidoComida, PedidoEncomienda y PedidoExpress; por ultimo, el paquete app donde se encuentra la clase Main

Instrucciones
1- Clone el repositorio desde Github https://github.com/rodrigoflo02/DOO-II---Semana2.git
2- Abrir el proyecto con Intellij Idea 
3- En menu, abrir el proyecto seleccionando la carpeta de este 
4- Si desea modificar el codigo debe editar la informacion base de las instancias (ejemplo: pedido1) en la clase Main respetando el tipo de datos requerido.
4- Ejecutar la clase Main.java desde el paquete UI (Ctrl + F5).
